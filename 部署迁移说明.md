# 从将仓库统一为 houlai_hysteresis

## 方案 A：直接在原仓库中替换文件

把本包中的以下文件上传到原 GitHub 仓库根目录并覆盖：

```text
app.py
requirements.txt
.streamlit/config.toml
README.md
Dockerfile
houlai_hysteresis_app.py
```

可以删除旧文件：

```text
早期临时主程序文件
早期临时启动脚本
```

然后回到 Streamlit Community Cloud，点击 app 的 Reboot 或 Redeploy。

## 方案 B：新建仓库

推荐新建仓库名：

```text
houlai_hysteresis
```

或：

```text
houlai-hysteresis
```

部署时填写：

```text
Repository: 你的用户名/houlai_hysteresis
Branch: main
Main file path: app.py
App URL: houlai-hysteresis-lc
```

## Streamlit 入口文件

入口文件始终填：

```text
app.py
```
